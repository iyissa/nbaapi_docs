# Introduction

As of the time of the time of writing, there are currently 135 different stats that this package can access from The NBA Website. (opens new window)

Each endpoint gives information about the data it returns, parameters that the endpoint takes and sample results.

Tip

To get all the arguments that an endpoint can return, use .expected_data.keys()

Example

from nba_api.stats.endpoints import commonplayerinfo
doncic_data = commonplayerinfo.CommonPlayerInfo(1629029)
doncic_data.expected_data.keys()

This returns all the arguments the commonplayerinfo endpoint can take.

dict_keys(['AvailableSeasons', 'CommonPlayerInfo', 'PlayerHeadlineStats'])

# All Time stats

# AllTimeLeadersGrids

This Endpoint returns All time leaders for different stats in the NBA.

from nba_api.stats.endpoints import alltimeleadersgrids
alltimeleadersgrids.AllTimeLeadersGrids()

# Query Parameters

API Parameter Name Python Parameter Variable Description Status Nullable
LeagueID (opens new window) league_id Options include NBA, ABA, G league, WNBA. Required Defaults to NBA.
PerMode (opens new window) per_mode_simple Options include total, per 36 minutes or per game Required Defaults to total.
SeasonType (opens new window) season_type Other options include regular season or preseason. Required Defaults to regular season.
TopX (opens new window) topx No info available Required No info available

# CommonPlayerInfo

# Description

This endpoint returns information about the players

from nba_api.stats.endpoints import commonplayerinfo
commonplayerinfo.CommonPlayerInfo(`parameter`)

# Query Parameters

API Parameter Name Python Parameter Variable Description Status Nullable
PlayerID (opens new window) player_id Player ID gotten from the static module Required No
LeagueID (opens new window) league_id_nullable League ID Optional Defaults to NBA. Other options include ABA, G league, WNBA.

# Arguments Taken

This Endpoint can take a number of arguments to refine the search.

['AvailableSeasons', 'CommonPlayerInfo', 'PlayerHeadlineStats']

# DraftHistory

# Description

This endpoint returns the entire history of the NBA Draft.

from nba_api.stats.endpoints import drafthistory

drafthistory.DraftHistory()

# Query Parameters

API Parameter Name Python Parameter Variable Description Required Nullable
LeagueID (opens new window) league_id League ID Required Defaults to NBA
TopX (opens new window) topx_nullable No Info available Optional Yes
TeamID (opens new window) team_id_nullable Team ID Optional Yes
Season (opens new window) season_year_nullable Options include current season, current datetime, current year of season. Optional Defaults to current season
RoundPick (opens new window) round_pick_nullable No Info available Optional Yes
RoundNum (opens new window) round_num_nullable No Info available Optional Yes
OverallPick (opens new window) overall_pick_nullable No Info available Optional Yes
College (opens new window) college_nullable No Info available Optional Yes

# Arguments Taken

This endpoint Only returns draft history with no other features.
